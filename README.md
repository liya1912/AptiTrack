# AptiTrack
Smart Online Aptitude &amp; Skill Evaluation System

## Project Description
AptiTrack is a smart web-based platform designed to conduct online aptitude and skill evaluation tests. 
It helps students practice aptitude questions and enables administrators and advisors to monitor 
performance and generate insightful reports.

## Technologies Used
Frontend: HTML, CSS, JavaScript, EJS  
Backend: Node.js, Express.js  
Database: MongoDB  

## Features

### Student Module
• User registration and secure login  
• Online aptitude and skill-based tests  
• Automatic evaluation and score generation  
• Performance tracking  

### Admin Module
• Manage users and roles  
• Create and manage tests and questions  
• View overall test analytics and results  

### Advisor Module
• Monitor assigned student performance  
• Receive automated performance reports via email  
• Reports are automatically sent once the test deadline is reached  
• Helps advisors track student progress without manual intervention
